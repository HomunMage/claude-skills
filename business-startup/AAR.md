# AAR 季度檢討表 (After Action Review)

## AAR 項目

| AAR項目 | 概念 |
|---------|------|
| 這個專案的目標是什麼？ | 確保所有人 view 一致 |
| 該專案是否符合我的業務核心價值觀？ | 確保所有人 view 一致, 個人成長 |
| 這為我的業務增添什麼具體價值？ | 個人成長 |
| 還有誰在做這樣的專案，我可以研究和參考？ | 競品, SWOT, 市場 |
| 我完成目標了嗎？ | 時程管理 |
| 哪部分進展順利？ | 增強的部分, 下一步行動細節 |
| 我會再做類似的事嗎？ | 延續上一題, 下一步行動細節 |
| 哪裡做得不好？ | 改變的部分, 下一步行動細節 |
| 下次我會採取什麼不同的做法？ | 延續上一題, 下一步行動細節 |
| 我從這個專案中學到了什麼？ | 新的結論，整體未來戰略調整方向 |
| 我可以採用哪些系統和流程來改進執行？ | 延續上一題，整體未來執行方針具體戰術 |
| 如何用更少的成本做到更高的成效 | 高效生產力, 優化思維 |

## 分類結構

### 核心價值 (確保所有人 view 一致)
- 這個專案的目標是什麼？
- 該專案是否符合我的業務核心價值觀？
- 這為我的業務增添什麼具體價值？

### 競品 (SWOT, 市場)
- 還有誰在做這樣的專案，我可以研究和參考？

### 戰略方向 (新的結論，整體未來戰略調整方向)
- 我完成目標了嗎？
- 我從這個專案中學到了什麼？
- 我可以採用哪些系統和流程來改進執行？
- 如何用更少的成本做到更高的成效

### 戰術執行 (用來規劃下一步行動細節)
- 哪部分進展順利？
- 我會再做類似的事嗎？
- 哪裡做得不好？
- 下次我會採取什麼不同的做法？
